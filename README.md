# libbase45

A simple Base45 library in C.

For details on the Base45 specification, see here: https://datatracker.ietf.org/doc/draft-faltstrom-base45/.

## Build instructions
Requires CMake.

    $ git clone https://github.com/kf03w5t5741l/base45.git
    $ cd base45
    $ mkdir build
    $ cd build
    $ cmake ..
    $ cmake --build .
